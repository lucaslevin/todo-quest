# Contributing to Todo Quest

Thanks for your interest in contributing to Todo Quest.

## Philosophy

Todo Quest aims to be a focused, high-quality productivity app.
Contributions should improve clarity, usability, performance, or maintainability.

Please keep changes:

- Small and focused
- Consistent with the existing architecture
- Aligned with the core vision of the project

## How to Contribute

1. Fork the repository
2. Create a feature or fix branch
3. Make your changes
4. Open a pull request with a clear description of what and why

## Code Style

- Follow the existing code style and conventions
- Prefer clarity over cleverness
- Avoid unnecessary abstractions
